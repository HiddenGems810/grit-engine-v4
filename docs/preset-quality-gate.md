# FORMAT Preset Quality Gate

FORMAT presets pass only when they behave like finished one-click products, not loose parameter stacks.

## Pass / Fail Criteria

- Pass only if the preset creates an obvious one-click improvement.
- Pass only if the preset identity is readable within 1 second.
- Pass only if the image still feels intentional and premium.
- Fail if skin looks waxy.
- Fail if melanated skin shifts gray, green, red, or orange unnaturally.
- Fail if shadows crush eyes, hair, beard, or facial structure.
- Fail if highlights clip forehead, cheeks, nose, teeth, or product edges.
- Fail if a portrait-safe preset changes identity.
- Fail if a product preset damages edge sharpness.
- Fail if a film preset looks like a flat yellow overlay.
- Fail if a viral preset looks cheap or dated.
- Fail if two presets are visually redundant.
- Fail if name and output do not match.
- Fail if extreme looks are not labeled extreme.
- Fail if hero presets are not useful on at least 70% of normal creator images.

## Required Test Image Classes

1. Studio portrait
2. Low-light phone selfie
3. Melanated skin portrait
4. Pale skin portrait
5. Beard/hair detail portrait
6. Makeup-heavy portrait
7. Product photo
8. Food photo
9. Streetwear/fashion photo
10. Bright outdoor lifestyle image
11. Night scene
12. AI-generated image with plastic skin
13. Compressed social media image
14. Flat/low-contrast photo
15. Oversaturated phone photo

## Scoring Rubric

- One-click improvement: 0-10
- Skin safety: 0-10
- Color taste: 0-10
- Detail preservation: 0-10
- Commercial usefulness: 0-10
- Distinctiveness: 0-10
- Modernity: 0-10
- Restraint: 0-10

A preset should not be hero-tier unless it averages 8+ across the rubric.
