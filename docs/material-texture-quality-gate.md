# FORMAT Material Texture Quality Gate

## Pass / Fail Rules

- Pass only if the texture feels integrated into the image.
- Fail if texture looks like a flat overlay.
- Fail if grain is uniform random noise.
- Fail if paper texture ignores tonal structure.
- Fail if halftone does not respond to image luminance.
- Fail if CMYK halftone lacks separation or misregistration behavior.
- Fail if Xerox looks like simple threshold only.
- Fail if film grain appears equally in highlights and shadows.
- Fail if texture damages eyes, teeth, skin, hairline, beard, jewelry, tattoos, or product edges.
- Fail if destructive materials are not labeled.
- Fail if remote texture failure breaks export.
- Fail if materials make premium presets look cheap.

## Required Test Image Classes

1. Studio portrait
2. Low-light portrait
3. Melanated skin portrait
4. Product photo
5. Food photo
6. Fashion photo
7. Bright outdoor image
8. Dark night image
9. Flat low-contrast image
10. AI-generated plastic-skin image
11. High-detail hair/beard image
12. Product with sharp edges
13. White-background ecommerce product

## Material Review Rubric

Score each material or preset-material pairing from 0-10:

- Physical believability
- Luminance response
- Detail preservation
- Skin safety
- Product edge safety
- Print or film specificity
- Restraint
- Commercial usefulness

A material should not be used by Signature or Portrait presets unless it averages 8+ on believability, restraint, and safety.
